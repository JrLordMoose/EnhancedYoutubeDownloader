Installation

Get started in minutes

1

Download Installer

Download the latest installer from the download section below or GitHub releases.

2

Run Installer

Double-click the installer and follow the setup wizard. Choose installation options like desktop shortcut.

3

Handle SmartScreen Warning

You may see a Windows SmartScreen warning. This is normal for unsigned installers. Click "More info" then "Run anyway".

The installer is safe - all code is open-source and can be reviewed on GitHub.
4

Launch & Enjoy

The app will launch automatically after installation. Start downloading your favorite YouTube content!
